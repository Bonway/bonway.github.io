海淘电商面试

发表于 2018-03-29 | 分类于 面试 | | 阅读次数
| 字数统计

导语

无意中，在Boss直聘中看到了一家招聘海淘母婴电商的公司，自己赶紧发了相对应的简历，期待着HR能给自己一个面试的机会。最主要的原因，莫过于自己之前做的APP绝大部分全是电商类的APP。同时自己也觉得，相比较普通的公司，自己是有与这家公司的HR多聊了几句的。别小看这几句，真的换来了所以所期望的，HR终于应了我，还约了面试的时间。

坐标

3月28号，大宁中心广场，之前没有来过，好一阵子难找，不过还是按照规定的时间来了。

面试前

来到这边公司，给人的感觉还行，一种活力十足的青春气息（对于1991年的我来说，可能我拉高了他们的平均年龄了吧）。HR短头发，看着很干练，说是她约的我，把我带到了一个小房间里面（旁边小房间貌似也在面试），让我现在这坐会，就离开了。
不一小会，又过来了一个HR，给我倒了一杯水，没注意她长啥样，只注意到抹着黑指甲油，不过这天气也确实有点热。

步入正题

（1）面试官小伙：

一个小伙抱着一台Mac就来了，看的出来，这应该就是我今天的面试官。（ps：不过，跟我想象中的还有点不太一样，因为没有笔试）
这个面试官看着年龄跟我差不多，没有多余的废话，然后就开始问了一些相关的问题。（努力回忆ing。。。）

  • 如何创建array，@[@”1”,@”2”,…]会不会崩溃，[NSMutableArray array] 和 [[NSMutableArray alloc]init]那个性能最高？
    不过自己确实没有深入的了解，只知道平时都用array创建。
  • if(){[self.delegate foo]},让我填if括号里面的，我自然说出了self.delegate res...:@selector(foo)，在iOS的传值方法中，自己用的最多的就是代理了，不过，平时都是打了几个字母就联想出来了，这个时候却说不出来，悲催。（ps：面试官小伙很6，居然能直接在他们的聊天系统上打印，没有联想哦~牛）
  • 用真机问了下首页的实现过程
  • 用真机看了下购物车的实现
  • runtime
    总结1：
    还有啥，想不起来了，好像也没啥了。自己感觉：最开始几个问题，是这个面试官最近开发中遇到的小问题吧，而了解相关的购物车模块，估计是他最近在实现的功能或者将要做的功能吧。
    没有问太多的第三方，也没有问相关的性能优化方面的，就连上架的流程等等都没有涉及到，总体说来，一般。

    （2）气质CTO：

    帅小伙说，要去叫他们老大。不一会，一个看似和蔼的CTO同样抱着Mac，进去了房间。浏览了相关的简历。
  • 指出了简历的的不合理的地方，不过自己之前也没有想到真会对自己大学所考的证书感兴趣。
  • 离职原因
  • 大学学的怎么样
  • 之前项目几人做，主要负责哪些？
  • TCP/UDP是什么？
  • 异或 1^1，1^0 的结果？ （ps：没答出来，忘记了，面试完赶忙看了下百度1^0=1,1^1=0）
  • 冒泡的时间复杂度？
    总结2：自我感觉答的很糟糕，大学有学到过，不过工作后还给老师了。诶~

    （3）漂亮HR：

    最后一个自然是HR了，大家都明白，HR想了解什么：上班时间，抗压能力，薪资待遇，离职原因，家庭环境，接下来的发展，有没有定居的计划，离职时间，社保等等。。。，不多说了，他们还有几个面试，然后就是等通知了。
    总结3：
    无论是HR面试还是之前的技术面，感觉一般，有些问题没答上，每个公司的问题都不一样，自己唯一能做的就是多了解，多做笔记，来弥补自己的不足吧。
